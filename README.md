MapLite
=======

Android Tile Map, fork from [osmdroid](http://code.google.com/p/osmdroid/)


Major differences from osmdroid
-------------
* Supports EPSG:4326 TileSystem
* Supports Google, ArcGIS Online and Tianditu Tile Source
